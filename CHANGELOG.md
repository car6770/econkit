# Changelog

## 1.0.0

### Added

- Macro risk screening
- Monetary policy stance analysis
- Business-cycle diagnosis
- Macro scenario simulation
- Markdown and JSON outputs
- CLI commands for validate, report, risk, policy, cycle, scenarios, and all
- Test suite for core analysis functions
- Python package configuration through `pyproject.toml`

### Notes

EconKit is an educational economics data analysis toolkit. Its analysis tools are not professional forecasts or official policy recommendations.
