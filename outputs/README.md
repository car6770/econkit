# Outputs Folder

This folder is reserved for files generated by EconKit examples and CLI commands.

Typical generated outputs include:

- Markdown reports
- CSV summary files
- JSON diagnostics
- PNG charts
- forecast tables
- scenario outputs

---

## Important

Generated files are usually not committed to Git.

The `.gitignore` file excludes most generated output folders.

This README exists only to explain the purpose of the folder.

---

## Example Commands

Generate a report:

```bash
python src/econkit_cli.py report data/sample_economic_data.csv --output-dir outputs/report
```

Generate all outputs:

```bash
python src/econkit_cli.py all data/sample_economic_data.csv --output-dir outputs --years 5 --stress-tests
```
