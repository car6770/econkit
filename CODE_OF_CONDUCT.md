# Code of Conduct

## Our commitment

EconKit is an open-source educational project for economics data analysis.

We are committed to providing a welcoming and respectful environment for students, researchers, and contributors of all backgrounds.

## Expected behavior

Contributors are expected to:

* Be respectful and constructive
* Use welcoming and inclusive language
* Give helpful feedback
* Focus on improving the project
* Be patient with beginners

## Unacceptable behavior

Unacceptable behavior includes:

* Harassment or personal attacks
* Discriminatory language
* Insulting or disrespectful comments
* Intentionally disruptive behavior
* Sharing private information without permission

## Project goal

The goal of EconKit is to make economics data analysis easier to learn and practice.

All discussions, issues, and contributions should support this educational purpose.
