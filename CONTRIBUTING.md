# Contributing to EconKit

Thank you for your interest in contributing to EconKit.

EconKit is an open-source Python toolkit for beginner-friendly economics data analysis. The goal of this project is to help students and early researchers learn how to clean, analyze, and visualize economic data using Python.

## Ways to contribute

You can contribute by:

* Adding new sample economic datasets
* Improving documentation
* Creating beginner-friendly tutorials
* Adding new analysis examples
* Improving Python functions in `src/econkit.py`
* Writing tests for existing functions
* Fixing bugs or improving code quality

## How to contribute

1. Open an issue describing the improvement or problem.
2. Fork the repository.
3. Create a new branch for your changes.
4. Make your changes.
5. Submit a pull request.

## Contribution guidelines

Please keep contributions beginner-friendly and clearly documented.

When adding code, try to include comments or explanations so that economics students who are new to Python can understand the workflow.

## Project goal

The main goal of EconKit is educational. Contributions should help make economics data analysis more accessible, practical, and easy to understand.
