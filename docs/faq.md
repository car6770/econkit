# EconKit FAQ

## What is EconKit?

EconKit is an open-source Python toolkit for economics data analysis.

It provides tools for:

- data loading
- data cleaning
- validation
- summary statistics
- charts
- macro diagnostics
- reporting
- scenario simulation
- forecasting
- lightweight regression

---

## Is EconKit only for macroeconomics?

The current version focuses mostly on macroeconomic examples.

However, the design can be expanded to other economics fields, such as:

- labor economics
- development economics
- international economics
- industrial organization
- public economics
- finance-related economic analysis

---

## Is EconKit beginner-friendly?

Yes.

EconKit is designed for students who are learning how to work with economic datasets.

The project emphasizes:

- readable code
- simple examples
- clear documentation
- transparent methods
- reproducible workflows

---

## Is EconKit a forecasting model?

No.

EconKit includes simple forecasting tools, but they are educational baseline methods.

They should not be interpreted as official forecasts.

---

## Does regression in EconKit prove causality?

No.

Regression describes statistical relationships in the dataset.

Causal interpretation requires research design.

---

## What data format does EconKit expect?

Most macro tools expect:

```text
year
gdp_growth
inflation_rate
unemployment_rate
interest_rate
```

---

## Can I use my own dataset?

Yes.

Your dataset should contain the required columns, or you should rename your columns before using EconKit's macro functions.

---

## Why does EconKit generate Markdown reports?

Markdown is:

- readable on GitHub
- easy to edit
- easy to version-control
- simple for beginners
- portable across tools

---
