# EconKit Tutorial

This tutorial explains how to use EconKit for a simple macroeconomic data analysis workflow.

## 1. Load the dataset

The sample dataset is located at:

```text
data/sample_economic_data.csv
```

It includes basic macroeconomic indicators such as GDP growth, inflation, unemployment, and interest rates.

## 2. Install requirements

Install the required Python packages:

```bash
pip install -r requirements.txt
```

## 3. Run the example

Run the example script:

```bash
python examples/analyze_macro_data.py
```

The script will:

* Load the sample dataset
* Print summary statistics
* Plot inflation over time
* Plot GDP growth over time

## 4. Interpret the results

Students can use the output to practice basic economic interpretation.

For example:

* Higher inflation may indicate rising price levels.
* GDP growth shows how the economy expands or contracts.
* Unemployment helps describe labor market conditions.
* Interest rates can be compared with inflation and growth patterns.
