# EconKit Roadmap

This roadmap describes possible future directions for EconKit.

The project goal is to become a beginner-friendly but professional economics data analysis toolkit.

---

## Version 1.x: Stability and Documentation

Current focus:

- stable Python API
- stable CLI workflow
- clear documentation
- reproducible examples
- GitHub Actions testing
- beginner-friendly usage instructions

Planned improvements:

- more tests for edge cases
- more sample datasets
- cleaner report templates
- improved chart customization
- stronger documentation examples

---

## Version 2.x: Applied Economics Workflows

Possible future features:

- panel-data helper functions
- time-series diagnostics
- event-study templates
- difference-in-differences examples
- simple causal inference teaching examples
- country-comparison workflows
- World Bank / FRED style data import examples

---

## Version 3.x: Research and Teaching Toolkit

Longer-term ideas:

- notebook tutorials
- course-ready exercises
- reproducible policy memo templates
- automated project scaffolding
- interactive dashboards
- stronger econometrics utilities

---

## Design Principles

Every new feature should be:

- clear
- tested
- documented
- beginner-friendly
- economically interpretable
- reproducible
- honest about limitations

EconKit should avoid becoming a black box.

The project should help users understand economic data, not just produce outputs.
